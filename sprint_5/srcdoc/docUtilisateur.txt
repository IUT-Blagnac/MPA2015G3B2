== Présentation 

=== Sujet :

Le but du projet OPTI est de fournir une application de gestion de projets tutorés informatiques.

=== Organisation du groupe :

Pour mener ce projet à bout, nous avons formé un équipe de 6 personnes. Nous allons utiliser la méthode agile SRCUM 
pour être le plus efficace possible. Chaque semaine, nous effectuons un sprint où chaque membre a une tâche assignée.
Nous devons fournir un recette à la fin de chaque sprint pour montrer notre avancée au client.

== Guide d'utilisation

Lancer l'application.

image::accueil.png[link="accueil.png",width="25%"]

Elle s'ouvre sur la liste des étudiants comme le montre l'image ci-dessus.
Vous pouvez naviguer entre les différents onglets pour accéder à la liste des intervenants ou des sujets.

Pour créer un nouveau sujet, étudiant ou intervenant, sélectionner le bon onglet puis cliquez sur le bouton "Ajouter".

image::creer.png[link="creer.png",width="25%"]

Une nouvelle page s'ouvre où vous pourrez saisir les informations demandés. Puis cliquer sur "Valider" pour confirmer.

image::supprimer.png[link="supprimer.png",width="25%"]

Si vous voulez supprimer une ligne ou plusieurs, sélectionnez sur la ou les lignes (elles seront bleutées) et cliquez sur le bouton "Supprimer".

Pour modifier, sélectionnez la case et réécrivez directement dans celle-ci.

image::sauver.png[link="sauver.png",width="25%"]

Si vous voulez enregistrer vos modifications, cliquez sur le bouton "Sauver" puis sur ok sur la fenêtre qui s'ouvre.

image::interface.png[link="interface.png",width="25%"]

Pour l'onglet "Projet", vous pouvez cliquer sur "Cloner" pour copier un projet déjà existant:

image::cloner.png[link="cloner.png",width="25%"]

Si vous cliquez sur l'onglet "A propos", vous aurez des informations sur le groupe et la création de l'application.

image::quitter.png[link="quitter.png",width="25%"]

Pour quitter l'application, cliquer sur "Quitter". La page ci-dessus s'ouvre et vous demande une confirmation.

WARNING: Si vous quittez sans avoir sauvegarder, vous perdrez vos modifications.
