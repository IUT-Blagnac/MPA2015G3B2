OPTIweb : Documentation Utilisateur

DUT/INFO/M3301
version 0.1
Table des matières

OPTIweb.html est une application HTML de consultation des projets tutorés gérés par OPTI.

OPTIweb.html est compatible avec l’ensemble des navigateurs et supports du moment (PC, tablettes, smartphones).

2. Fonctionnalités disponibles - SPRINT 4

Application

  • En tant qu’internaute, je veux accéder à l’application OPTIweb depuis tout support fixe ou mobile

  • En tant qu’internaute, je veux pouvoir consulter les projets, les sujets, les étudiants et les intervenants dans des pages séparées

  • En tant qu’internaute, je veux pouvoir consulter le nom des développeurs de l’application, le nom du Product Owner ainsi que les technologies utilisées

Page Projets

  • En tant qu’internaute, je veux pour chaque projet en voir le nom, le titre, le client, le superviseur, le groupe et les étudiants réalisant le projet

  • En tant qu’internaute, je veux pouvoir rechercher un projet dans la liste des projets en fournissant une chaine de caractère quelconque

Page Sujets

  • En tant qu’internaute, je veux pour chaque sujet en voir le nom, le titre et le ou les noms des groupes d'étudiants réalisant le sujet

  • En tant qu’internaute, je veux pouvoir rechercher un sujet dans la liste des sujets en fournissant une chaine de caractère quelconque

  • En tant qu’internaute, je veux pouvoir sélectionner un sujet et voir les projets réalisant le sujet

Page Etudiants

  • En tant qu’internaute, je veux pour chaque étudiant en voir le nom, le prénom ainsi que son groupe

  • En tant qu’internaute, je veux pouvoir rechercher un étudiant dans la liste des étudiants en fournissant une chaine de caractère quelconque

  • En tant qu’internaute, je veux pouvoir sélectionner un étudiant et voir le projet qu’il réalise

Page Intervenants

  • En tant qu’internaute, je veux pour chaque intervenant en voir le nom, le prénom ainsi que le nombre de ses interventions en tant que Client ou Superviseur

  • En tant qu’internaute, je veux pouvoir rechercher un intervenant dans la liste des intervenants en fournissant une chaine de caractère quelconque

  • En tant qu’internaute, je veux pouvoir sélectionner un intervenant et voir le(s) projet(s) où il intervient

3. Comment lancer l’application OPTIweb

  1. Sur tout support, charger l’url suivante dans un navigateur http://algec.iut-blagnac.fr/~jmi/MPA/OPTIweb.html

L’application OPTIweb.html nécessite une connexion Internet lors de son lancement.

Une fois chargée dans un navigateur, aucune communication réseau n’est éffectuée : vous pouvez alors travailler hors connexion.

4. Comment sauvegarder l’application OPTIweb

  1. Enregistrer le fichier OPTIweb.html sur votre support (environ 30Ko)

  2. Charger le fichier OPTIweb.html local dans un navigateur pour lancer l’application (une connexion réseau n’est requise qu’au chargement).

5. OPTIweb : Page d’Accueil

La Page d’Accueil permet d’accéder :

  1. aux Projets tutorés de l’année

  2. aux Sujets de l’année

  3. aux Etudiants du DUT INFO S3 de l’année

  4. aux Intervenants de l’année

    Le Bouton permet d’obtenir des informations sur l’application OPTIweb

6. OPTIweb : Page Projets

La Page Projets présente par défaut l’ensemble des projets tutorés de l’année en indiquant leur nom, leur titre, leur client, leur superviseur et leur groupe d'étudiants.

6.1. OPTIweb : Rechercher un projet

Pour rechercher un projet particulier :

  1. Saisir une chaine dans le champ de recherche

  2. L’ensemble des projets contenant la chaine saisie est alors fourni

    Le Bouton permet de revenir à la Page d’Accueil de l’application OPTIweb

7. OPTIweb : Page Sujets

La Page Sujets présente par défaut l’ensemble des Sujets de l’année en indiquant leur nom, leur titre et le ou les groupe(s) d'étudiants qui le réalise(nt).

7.1. OPTIweb : Sélectionner un sujet

  1. Dans la liste des sujets, Cliquer sur la ligne correspondant au sujet choisi

  2. L’ensemble des projets réalisant le sujet sélectionné est alors fourni

    Le Bouton permet de revenir à la Page Sujets de l’application OPTIweb

7.2. OPTIweb : Rechercher un sujet

Pour rechercher un sujet particulier :

  1. Saisir une chaine dans le champ de recherche

  2. L’ensemble des sujets contenant la chaine saisie est alors fourni

Le Bouton permet de revenir à la Page d’Accueil de l’application OPTIweb

8. OPTIweb : Page Etudiants

La Page Etudiants présente par défaut l’ensemble des Etudiants de l’année en indiquant leur nom, leur prénom et leur groupe.

8.1. OPTIweb : Sélectionner un étudiant

  1. Dans la liste des étudiants, Cliquer sur la ligne correspondant à l'étudiant choisi

  2. Le projet réalisé par l'étudiant sélectionné est alors fourni

Le Bouton permet de revenir à la Page Etudiants de l’application OPTIweb

8.2. OPTIweb : Rechercher un étudiant

Pour rechercher un étudiant particulier :

  1. Saisir une chaine dans le champ de recherche

Pour Effacer la sélection

  • Cliquer sur la X du champ de recherche

  1. L’ensemble des étudiants contenant la chaine saisie est alors fourni

8.3. OPTIweb : Rechercher un groupe

Pour rechercher un groupe particulier :

  1. Saisir 'groupe ' suivi du nom du groupe dans le champ de recherche

  2. L’ensemble des étudiants du groupe saisi est alors fourni

Le Bouton permet de revenir à la Page d’Accueil de l’application OPTIweb

9. OPTIweb : Page Intervenants

La Page Intervenants présente par défaut l’ensemble des Intervenants de l’année en indiquant leur nom, leur prénom et le nombre de fois où ils sont Client ou Superviseur.

9.1. OPTIweb : Sélectionner un Intervenant

  1. Dans la liste des intervenants, Cliquer sur la ligne correspondant à l’intervenant choisi

  2. La liste des projets où l’intervenant sélectionné participe est alors fournie

Le Bouton permet de revenir à la Page Intervenants de l’application OPTIweb

9.2. OPTIweb : Rechercher un Intervenant

Pour rechercher un intervenant particulier :

  1. Saisir une chaine dans le champ de recherche

  2. L’ensemble des intervenants contenant la chaine saisie est alors fourni

Pour Effacer la sélection

  • Cliquer sur la X du champ de recherche

Le Bouton permet de revenir à la Page d’Accueil de l’application OPTIweb
